== Description ==

Create 'Store Credit' coupons for customers which are redeemable at checkout.
Visit our [product page](https://woocommerce.com/products/store-credit/) for more info.

== Minimum Requirements ==

* PHP version 5.6 (PHP 7.4 or greater is recommended)
* MySQL version 5.0 (MySQL 5.7 or greater is recommended)

== Installation ==

1. Unzip and upload the plugin’s folder to your /wp-content/plugins/ directory.
2. Activate the extension through the ‘Plugins’ menu in WordPress.
3. Go to WooCommerce > Settings > Store Credit to configure the plugin.
